# GitHub Profile Card

Современное веб-приложение для просмотра и сравнения GitHub-профилей.

Проект позволяет находить пользователей GitHub по username, просматривать их статистику, анализировать репозитории, определять наиболее используемые языки программирования, сравнивать двух пользователей между собой и сохранять историю поиска.

## Используемые API

Получение информации о пользователе:

```http
https://api.github.com/users/USERNAME
```

Получение списка репозиториев:

```http
https://api.github.com/users/USERNAME/repos
```

## Как запустить локально

1. Клонировать репозиторий:

```bash
git clone https://github.com/ocheri-tdm/github-profile-card.git
```

2. Перейти в папку проекта:

```bash
cd github-profile-card
```

3. Открыть файл `index.html` в браузере.

Либо использовать расширение Live Server в VS Code.

## Функционал

### Основной функционал

- Поиск GitHub-пользователя по username
- Поиск по кнопке «Найти»
- Поиск по нажатию клавиши Enter
- Отображение аватара пользователя
- Отображение имени пользователя
- Отображение биографии
- Отображение количества публичных репозиториев
- Отображение количества подписчиков
- Отображение количества подписок
- Загрузка последних 5 репозиториев
- Переход в репозиторий по клику
- Индикатор загрузки
- Обработка ошибок API
- Сообщение при отсутствии пользователя

### Дополнительный функционал

- Подсчёт популярных языков программирования пользователя
- Сортировка репозиториев:
  - по количеству звёзд
  - по дате создания
  - по последнему обновлению
- Сравнение двух GitHub-пользователей
- История последних 5 поисковых запросов
- Сохранение истории через LocalStorage
- Быстрый доступ к ранее найденным пользователям
- Кнопка случайного пользователя
- Светлая тема
- Тёмная тема
- Неоновый интерфейс
- Анимированный фон
- Адаптивная верстка для мобильных устройств

## Обработка ошибок

Приложение корректно обрабатывает:

- несуществующий username
- отсутствие интернет-соединения
- ошибки GitHub API
- пустой ввод
- превышение лимита GitHub API

Пример сообщения:

```text
Пользователь не найден
```
